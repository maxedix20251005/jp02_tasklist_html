# Data Reset Instructions (EN/JA)

## English
### Option A: DevTools
1. Open DevTools (F12).
2. Application -> IndexedDB -> tasklist_db.
3. Delete the database or clear the stores.
4. Reload the app to reseed defaults.

### Option B: Clear site data
1. Chrome Settings -> Privacy and Security -> Clear browsing data.
2. Clear site data for file:// or localhost origin.
3. Reload the app.

## 日本語
### 方法A: DevTools
1. DevTools を開く（F12）。
2. Application -> IndexedDB -> tasklist_db。
3. DB 削除またはストアをクリア。
4. リロードして初期データを再投入。

### 方法B: サイトデータ削除
1. Chrome 設定 -> プライバシーとセキュリティ -> 閲覧データ削除。
2. file:// や localhost のサイトデータを削除。
3. アプリを再読み込み。
