# Security & Privacy Notes (EN/JA)

## English
- All data stays in the local browser (IndexedDB).
- No data is sent to any server.
- Clearing browser data deletes all tasks.
- This app does not implement authentication.

## 日本語
- データはブラウザ内（IndexedDB）にのみ保存されます。
- サーバ送信は行いません。
- ブラウザデータ削除でタスクは消えます。
- 認証機能はありません。
