# Limitations (EN/JA)

## English
- No backend. Multi-user collaboration is not supported.
- Data is stored per browser/profile. Not shared across devices.
- Clearing site data removes all tasks.
- Different origins (file:// vs http://localhost) use different databases.
- No authentication or access control.

## 日本語
- バックエンドがないため、複数人同時利用はできません。
- データはブラウザ/プロファイル単位で保存され、端末間共有はされません。
- サイトデータを削除すると全タスクが消えます。
- origin が違うと別の DB になります（file:// と http://localhost）。
- 認証やアクセス制御はありません。
